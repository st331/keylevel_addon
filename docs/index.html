🔑 KeyLevelLogs

Paste your Mythic+ applicants — see their Warcraft Logs percentiles for your key level: any dungeon at that level, and your dungeon at the same level, a higher one, or one below.

Setup — Warcraft Logs API credentials ()
  1. Open your API client at warcraftlogs.com/api/clients (the client id below is pre-filled for the “MPlus Dashboard” client — or create your own free client; “Public Client” unchecked).
  2. Copy the client secret from that page and paste it here. Credentials are stored only in this browser (localStorage) and sent only to warcraftlogs.com — never put the secret anywhere else.

Colors follow Warcraft Logs tiers. “(higher)” = they logged the dungeon above your level (counts at least as much); “(one below)” = only one level below. A player with no rows simply doesn’t log — same answer the website would give.
